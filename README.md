# Features

# Tech Stack

* React JS
* Easy Peasy

# How to run this

* Open your terminal (NOTE: git bash prefered/ Terminal on VS Code)
* Navigate to the project folder where you have cloned this repo.
* Run the local frontend server: `yarn start`
